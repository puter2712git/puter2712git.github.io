## 인라인(inline) 함수

일반적인 함수의 호출은 다음과 같이 이루어진다.

1. 특정 함수를 만난다.
2. 해당 함수의 주소로 프로그램을 점프시킨다.
   - 함수를 실행하는 줄 다음 행의 주소를 메모리에 저장
   - 스택에 매개변수 복사
3. 함수의 처리가 종결되면 다시 원래의 주소로 돌아온다.
   - 함수의 리턴값 레지스터에 복사
   - 저장해 두었던 주소로 돌아옴

이처럼 함수를 사용하는 데 시간이 소요되는데, 이 시간을 줄이기 위해 도입된 것이 인라인 함수이다.

인라인 함수는 컴파일된 함수 코드가 프로그램의 다른 코드 안에 직접 삽입되어 있다. <br>
함수를 그 자리에서 처리하므로, 프로그램의 주소가 점프하지 않는다. <br>
속도 측면에서는 인라인 함수가 유리하지만, 메모리 측면에서는 불리하다.

따라서 인라인 함수는 함수 코드 자체를 수행하는 데 걸리는 시간이 함수 호출의 시간보다 짧을 때만 사용하는 것이 좋다.

인라인 함수를 사용하려면 함수의 선언 또는 정의에서 앞에 `inline` 키워드를 붙여야 한다. <br>

> 재귀함수는 인라인화 되지 않는다.

<br>

### inline.cpp
```cpp
// inline.cpp -- 인라인 함수 사용하기
#include <iostream>

// 인라인 함수 정의
inline double square(double x)
{
  return x * x;
}

int main()
{
  using namespace std;

  double a, b;
  double c = 13.0;

  a = square(5.0);
  b = square(4.5 + 7.5);
  cout << "a = " << a << ", b = " << b << endl;
  cout << "c = " << c;
  cout << ", c의 제곱 = " << square(c++) << endl;
  cout << "현재 c = " << c << endl;

  return 0;
}
```

> 매크로 기능으로도 인라인 함수를 비슷하게 만들 수 있다. <br>
> `#define SQUARE(X) X * X`
>
> 매크로는 함수에 매개변수를 넘기는 것 대신 텍스트를 대체함으로써 동작한다.
>
> 하지만 매크로는 값을 전달하지 않는 문제가 있다. <br>
> `d = SQUARE(c++);`와 같은 구문은 `c`를 두 번 증가시킨다.
